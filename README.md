# Subscription Tracker AI MCP

Track recurring subscriptions, calculate monthly and yearly totals, and identify upcoming renewals.

## Tools

- `add_subscription` — Add a new subscription with billing cycle.
- `list_subscriptions` — List all tracked subscriptions.
- `calculate_totals` — Calculate monthly and yearly spend.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m subscription-tracker-ai-mcp.server
```
